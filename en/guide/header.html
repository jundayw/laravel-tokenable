# Specify Response Structure

WARNING

Authorization: Basic <base64(hash:access_token)>

Use withToken method to specify the response structure type:

return $this->guard('web')
    ->login($user)
    ->withToken('hash')
    ->createToken(name: 'PC Token', platform: 'pc');
